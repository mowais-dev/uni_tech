## 🚀 Features

- **Cutting Edge UI**: Modern, responsive, and visually appealing interface for a seamless user experience.
- **Stripe Payment Integration**: Secure and fast payment processing using Stripe.
- **Firebase Backend**: Real-time database, authentication, and cloud functions powered by Firebase.
- **Web Application**: Built with Flutter for the web, ensuring cross-platform compatibility and performance.
- **Modular Architecture**: Clean code structure with providers, services, and models for easy maintenance and scalability.

## ⚡ Getting Started

1. **Clone the repo**
2. **Install dependencies**
	```sh
	flutter pub get
	```
3. **Run on Web**
	```sh
	flutter run -d chrome
	```
4. **Configure Firebase & Stripe**
	- Update your `firebase_options.dart` and `stripe_config.dart` with your keys.
